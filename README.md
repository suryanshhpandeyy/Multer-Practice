# 📁 Multer File Upload Practice

A simple file upload application built using **Node.js**, **Express.js**, and **Multer**. This project demonstrates how to handle multipart form data and upload files to the server efficiently.

## 🚀 Features

- Upload files through a web form
- Store uploaded files on the server
- Handle multipart/form-data requests using Multer
- Simple and clean Express.js backend
- Beginner-friendly project structure

## 🛠️ Tech Stack

- Node.js
- Express.js
- Multer
- EJS

## 🎯 Learning Outcomes

- Understanding Express.js middleware
- Handling file uploads with Multer
- Processing multipart/form-data requests
- Managing server-side file storage
- Building backend functionality with Node.js
